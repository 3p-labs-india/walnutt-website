## PROMPT 1 — OVERALL SITE STRUCTURE

Create a two-page marketing website for "Walnutt" — a career accelerator platform for tech engineers. Confident, peer-level tone — think developer tool, not job board. Sage green (#3A6B4C) primary accent on warm off-white (#FAFBF9) background.

Font system: Inter for all text. Weight 800 for H1 (56px), weight 700 for H2 (40px), weight 600 for H3 (24px), weight 400 for body (17px). JetBrains Mono 13px only inside browser mockup illustrations for URLs.

**Page 1 — For Engineers.** Three sections: 1) Hero — large all-caps headline "CAREER ACCELERATOR FOR TECH TALENT" with a cycling overline above showing different roles, subheadline, primary CTA button, and micro-copy. 2) "How It Works" — overline label, H2, three horizontal tab pills (01 Assess, 02 Build, 03 Stand Out), each tab shows left-side text (H3 + body paragraph) and right-side browser mockup card. 3) "Built For Every Stage" — overline, H2, body text, a horizontal row of role tags as pills, then a 2×3 grid of skill dimension cards (icon + title + one-line description), ending with a CTA button.

**Page 2 — For Companies.** Three sections: 1) Hero — large all-caps headline "YOUR NEXT ENGINEER IS ALREADY VERIFIED" with subheadline, primary and secondary CTA buttons. 2) "How It Works" — same tab structure as engineers page (01 Post, 02 Review, 03 Hire) with left text and right browser mockup. 3) "We Handle The Hard Parts" — overline, H2, body paragraph, three side-by-side value columns (Assessed / Interviewed / Matched) each with a title and description, a subtle pricing line below, and two CTA buttons.

Both pages share: sticky top nav with Walnutt logo left, a pill-shaped "For Engineers" / "For Companies" toggle center (active state: #3A6B4C background white text, inactive: white background #6B7280 text, 1px #D1D5DB border), and "Take the Assessment" CTA button right in #3A6B4C. Minimal footer: Walnutt logo, "by 3P Labs Private Limited", © 2026 3P Labs Private Limited.

Max content width 1120px centered on 1440px canvas. 96px vertical gap between every section on desktop, 64px on mobile. All cards 14px border-radius with layered shadow. Fully responsive: single-column stack on mobile (375px), 8-column grid on tablet (768px), 12-column on desktop. Nav becomes hamburger on mobile. Every element fades up on scroll entry (opacity 0→1, translateY 16px→0, 350ms ease-out, children staggered 80ms).

## PROMPT 4 — BRAND & VISUAL SYSTEM

Apply this exact design system across the entire Walnutt site:

**Colors** — Primary sage #3A6B4C: used only on primary CTA buttons, active tab pills, the accent words in hero headlines ("TECH TALENT" and "VERIFIED"), progress bars inside mockups, and top-border accents on value columns. Maximum 3 sage elements visible per viewport. Sage dark #2D5A3D: hover states for sage buttons and text color inside sage-light tags. Sage light #E8F0EB: skill tag pill backgrounds and role tag pill backgrounds only. Black #1A1A1A: all headlines (H1, H2, H3). Gray-800 #374151: body text. Gray-500 #6B7280: secondary text, overlines, inactive tab text, dimension card descriptions. Gray-400 #9CA3AF: placeholder text, micro-copy. Gray-300 #D1D5DB: borders, dividers, inactive tab borders. Gray-100 #F3F4F6: not used as section background — keep all sections on #FAFBF9 for simplicity. White #FFFFFF: cards, inputs, browser mockups. Off-white #FAFBF9: page background for both pages.

**Typography** — Inter everywhere. H1: 56px weight 800, line-height 1.05, tracking -0.02em, uppercase (hero only, one per page). H2: 40px weight 700, line-height 1.1, tracking -0.01em, sentence case. H3: 24px weight 600, line-height 1.3. Body: 17px weight 400, line-height 1.6. Small/micro-copy: 13px weight 400. Overlines: 12px weight 500, uppercase, tracking 0.1em. Button labels: 15px weight 600. JetBrains Mono 13px only inside browser mockup top bars.

**Spacing** — 8px base grid. 96px between all major sections (64px mobile). 48px between sub-blocks within a section. 24px between heading and body text. 32px card internal padding (24px mobile). 16px gap between side-by-side buttons. 8px gap between tab pills and between role tag pills.

**Components** — Primary button: #3A6B4C bg, white text, 15px/600, 14px 28px padding, 10px radius. Hover: #2D5A3D, translateY(-1px), shadow 0 4px 12px rgba(58,107,76,0.25), 180ms ease-out. Secondary button: white bg, 1.5px #D1D5DB border, #1A1A1A text, same padding and radius. Hover: border #3A6B4C, bg #E8F0EB, 180ms. Tab pill active: #3A6B4C bg, white text, 100px radius, 10px 20px padding. Tab pill inactive: white bg, 1px #D1D5DB border, #6B7280 text, same shape. Browser mockup: white bg, 1px rgba(0,0,0,0.06) border, 14px radius, shadow 0 8px 32px rgba(0,0,0,0.08) and 0 2px 8px rgba(0,0,0,0.04). Skill tag: #E8F0EB bg, #2D5A3D text, 13px/500, 5px 10px padding, 6px radius. Dimension card: white bg, 1px #E5E7EB border, 12px radius, 24px padding, hover lifts 2px with shadow.

**Motion** — Scroll entrance: every element fades up (opacity 0→1, translateY 16px→0) at 350ms ease-out, IntersectionObserver threshold 0.15, stagger children 80ms. Tab content: 200ms cross-fade. Button hover: 180ms ease-out. Cycling overline: 300ms slide-up with fade, 3s hold per item. No complex hero animations, no parallax, no mesh gradient animation. Respect prefers-reduced-motion: disable all animation.
